## INSTRUCTIONS

1. Install node modules

```npm install```

2. Run the SCSS styles

```sass assets/scss/main.scss assets/css/styles.css```

- for automatic watch on scss changes 

```sass --watch assets/scss/main.scss:assets/css/styles.css```
